# 🎵 Dynamic Weather Beats

**「世界のどこかへ、音のトリップ。」**
現在の天気や時間に合わせて、あなただけのプレイリストをお届けするWebアプリケーションです。

## 📖 サイトのコンセプト (About this App)
窓の外の景色が変われば、聴きたい音楽も変わる。
**Dynamic Weather Beats**は、あなたが今いる場所、あるいは遠く離れた憧れの都市の「今の空気」を読み取り、そのシチュエーションに最もマッチした音楽をSpotifyから自動でセレクトします。

晴れた日の昼下がりには軽快なポップスを。
雨の降る夜には、静かなジャズやローファイ・サウンドを。
同じ「晴れの日の昼」でも、アクセスするたびにランダムで新しい曲に出会える一期一会のミュージック体験を提供します。気に入った選曲は、ワンクリックであなたのSpotifyにプレイリストとして保存可能です。

## ✨ 主な機能 (Features)
* **🌤 天気・時間帯との連携**: OpenWeatherMap APIを使用し、指定した都市のリアルタイムな天候と現在時刻を取得。
* **🎧 ムードに合わせた自動選曲**: 取得した環境データ（晴れ/雨/雪、朝/昼/夕方/夜など）に基づき、Spotify Web APIで最適な楽曲を8曲ピックアップ。
* **🎲 ランダム選曲アルゴリズム**: 同じ環境条件でも、アクセスするたびに異なるキーワードプールとオフセットを使用し、常に新しい楽曲との出会いを創出。
* **💾 プレイリスト保存機能**: 気に入った楽曲リストを、自分のSpotifyアカウントに直接プレイリストとして保存可能。
* **🎨 ダークモダン・UIデザイン**: 音楽への没入感を高める、深いネイビーを基調とした美しいグラデーションとグラスモーフィズムデザイン。

## 🛠 使用技術 (Tech Stack)
* **Frontend**: HTML5, CSS3, Vanilla JavaScript
* **APIs**: 
  * [Spotify Web API](https://developer.spotify.com/documentation/web-api/) (OAuth 2.0 PKCE 認証, 楽曲検索, プレイリスト作成)
  * [OpenWeatherMap API](https://openweathermap.org/api) (現在地・指定都市の気象データ取得)

## 🚀 使い方 (How to Use)
1. トップページで **「Spotifyで始める」** をクリックし、お持ちのSpotifyアカウントでログイン（連携承認）します。
2. ログイン後、以下のいずれかの方法で都市を選択します。
   * **Explore Cities**: 画面上の美しい都市カード（Paris, Tokyo, Londonなど）をクリック。
   * **検索ボックス**: 気になる都市名をローマ字（例: Kyoto, Seattle）で入力して「検索」をクリック。
   * **📍現在地**: デバイスのGPS機能を利用して、あなたが今いる場所の天気を取得。
3. 気候データに基づいたダッシュボードと、あなたのためだけのプレイリストが表示されます。
4. **「🎵 プレイリストとして保存」** ボタンを押すと、あなたのSpotifyアカウントにプレイリストが自動作成されます。
